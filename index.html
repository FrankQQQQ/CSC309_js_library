Getting started with Calendarrr !

This is a javascript library that helps developers deal with dates and calendars, and helps students(end-users) deal with their assignment due dates.

How to use this library

This is a library that based on javascript.

In order to use this library, users need to create a new Calendarrr class in js file and call this class's functions.

e.g. let my_calendar = new Calendarrr()



Available Operations

_____________________________________________________________________

1. isLeapYear(year)

Checking whether a year is leap year

_____________________________________________________________________

2. getBumberOfDates(year, month)

Calculating how many dates are there in this month.

_____________________________________________________________________

3. isValidDate(date)

Checking whether a date is a valid date

_____________________________________________________________________

4. checkDay(date)

Checking what day is it of the date.

e.g.
2020, March 5th is Thursday
2020, March 7th is Saturday
2024, April 7th is Sunday

_____________________________________________________________________

5. getNextMonth(year, month) / getLastMonth(year, month)

getting next month/last month

_____________________________________________________________________

6. numToEng(month)

Transferring month number into English

_____________________________________________________________________

7. checkMonth(year, month)

Displaying this month's calendar in list mode. Note that the first index of each list represents Sunday

_____________________________________________________________________

8. checkYear(year)

Checking this year's calendar in list mode.

11. searchDateDay(date, day, year)
